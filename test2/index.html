Test 2

Hover over an image to make it clear and click on an image to keep it clear.

This page uses the CSS 'filter' property, ':hover', and "onclick". As with Test 1, the hover rule makes images clear when the mouse is over the image. However, this test also allows for images to style unblurred by toggling their CSS class when clicked.
